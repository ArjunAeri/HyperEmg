This system is designed to offer a streamlined hyperparameter optimization of a convolutional neural network and then result visualization via tensorboard for the following dataset:

https://archive.ics.uci.edu/ml/machine-learning-databases/00481/EMG_data_for_gestures-master.zip

(8 channel emg with 7 different classes)

Instructions:

Start jupyter notebook in the project directory and then read the documentation on the emgb notebook and run.
